# tic-tac-toe

## Live Demo: https://sophlanz.github.io/tic-tac-toe/

## Description 
A simple tic-tac-toe game for 2 players with creative UI. 

## Tools and Technologies
- Javascript 
- HTML5
- CSS3

## Features
- Choose player names. 
- Animation that displays player turn. 
- A pop-up screen that displays the winner.
